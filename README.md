# Face Mask Detection System 

A computer vision-based face mask detection system developed using Python, OpenCV, and Haar Cascade classifiers. The project detects faces in real-time and predicts whether a person is wearing a mask or not.

---

##  Features

- Real-time face detection using Haar Cascade
- Mask / No-Mask classification
- Image-based and webcam-based detection
- Deep learning model integration using Keras
- Easy-to-use Jupyter Notebook implementation

---

##  Technologies Used

- Python
- OpenCV
- TensorFlow / Keras
- Haar Cascade Classifier
- Jupyter Notebook
- NumPy
